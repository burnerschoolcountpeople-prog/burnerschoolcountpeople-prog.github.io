# burnerschoolcountpeople-prog.github.io
A static website that displays live room occupancy using data from Supabase.It fetches the latest {room_id, timestamp, people_count} per room (read-only via anon key), renders a responsive grid, and supports manual refresh and optional realtime updates. Built with HTML/CSS/JS + Supabase JS SDK. Designed to be fast, minimal and maintainable.
